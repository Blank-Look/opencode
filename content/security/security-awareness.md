# Security Awareness

Staff training, phishing simulations, and security culture programs.

## Training Program

| Module | Frequency | Audience |
|---|---|---|
| Security Essentials | Annually | All staff |
| Phishing Awareness | Quarterly | All staff |
| Data Protection | Annually | All staff |
| Secure Development | Onboarding | Developers |

## Phishing Simulations

- Conducted monthly via KnowBe4 platform
- Results tracked by department
- Repeat offenders assigned additional training
- Reports reviewed by security committee

## Reporting

- Report phishing: `phishing@contoso.com`
- Report security incident: `security@contoso.com`
- Emergency: Call SOC at extension 555-0199
