# Procurement

Standard process for acquiring ICT assets.

## Procurement Workflow

```
Request → Approve → Quote → Order → Receive → Register
```

### 1. Request

- Submit procurement request via service portal
- Include: item description, quantity, business justification
- Attach any quotes or references

### 2. Approval

| Value | Approver | SLA |
|---|---|---|
| < $500 | ICT Manager | 1 day |
| $500 - $5,000 | ICT Manager + Budget Holder | 3 days |
| > $5,000 | ICT Manager + Budget Holder + Exec | 5 days |

### 3. Quotation

- Obtain minimum 2 quotes for items > $1,000
- Preferred vendors list reviewed annually
- Consider TCO (including support, licensing, training)

### 4. Ordering

- Raise purchase order per finance process
- CC ICT team on order confirmation
- Record estimated delivery date

### 5. Receiving

- Verify items against delivery note
- Check for damage
- Report discrepancies within 2 business days

### 6. Registration

- Create asset record in the register
- Assign asset ID (physical label)
- Update financial system

## Software Procurement

For SaaS and software licences:
- Review licensing terms and data processing agreement
- Ensure GDPR/dpa compliance (DPA signed)
- Record: licence count, renewal date, cost centre
- Set calendar reminder for renewal review (60 days before)
