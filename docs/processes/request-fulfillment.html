Request Fulfillment

Handle standard service requests from users in an efficient and consistent manner.

Service Request Catalogue

Request Category SLA Cost Centre
New user account setup Onboarding 4 hours HR
Software installation Software 1 business day Department
Hardware request (laptop) Hardware 2 business days Department
VPN access Access 4 hours Manager approval
Password reset Access 1 hour —
File share access Access 4 hours Data owner approval

Workflow

1. Submission

  • User submits request via the service portal
  • Request is automatically categorised
  • Required approvals identified

2. Approval

Request Type Approver Method
Account creation Manager Email approval
Hardware purchase Manager + Budget holder Procurement system
Software install Team Lead Auto-approved (standard software)
Access changes Data owner Ticketing system

3. Fulfillment

  • Assign to appropriate team member
  • Execute per the relevant runbook
  • Update request with completion notes

4. Closure

  • Notify user that request is complete
  • Confirm satisfaction
  • Close the ticket

Self-Service

Users are encouraged to use the service portal for:

  • Password resets (self-service via MFA)
  • Software catalogue requests
  • Status checks on existing tickets
  • Knowledge base searches
