# Asset Life Cycle Overview

End-to-end management of ICT assets from identification through procurement, deployment, maintenance, and disposal.

## Asset Life Cycle Phases

```
Procurement → Deployment → Operation → Maintenance → Disposal
```

## What is an ICT Asset?

Any item of value to the organisation that requires management, including:

| Category | Examples |
|---|---|
| Hardware | Servers, laptops, switches, firewalls, peripherals |
| Software | Licences, SaaS subscriptions, open-source dependencies |
| Network | IP addresses, VLANs, certificates, DNS zones |
| Cloud | Cloud accounts, reserved instances, storage buckets |
| Data | Databases, file shares, backup media |

## Asset Register

All assets are recorded in the central asset register with:

| Field | Description |
|---|---|
| Asset ID | Unique identifier (ICT-XXXX) |
| Category | Hardware / Software / Network / Cloud / Data |
| Status | Ordered / In use / Maintenance / Retired |
| Location | Physical location or cloud region |
| Owner | Responsible team member |
| Purchase Date | Date of acquisition |
| Warranty End | Warranty / support expiry |
| EOL Date | End-of-life / end-of-support date |
| Value | Purchase cost or subscription cost |
