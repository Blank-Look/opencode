Threat Management

Detection, analysis, and remediation of security threats across the ICT environment.

Threat Detection Tools

Tool Purpose Coverage
Microsoft Defender Endpoint detection and response All managed endpoints
Microsoft Sentinel SIEM and SOAR Cloud and on-premises
Intrusion Detection Network traffic analysis Core network segments

Incident Response Process

  1. Identify — Detect and validate the threat
  2. Contain — Isolate affected systems
  3. Eradicate — Remove the threat
  4. Recover — Restore normal operations
  5. Learn — Conduct post-incident review

Response Tiers

Tier Response Time Severity
Tier 1 15 minutes Critical
Tier 2 1 hour High
Tier 3 4 hours Medium
Tier 4 Next business day Low
